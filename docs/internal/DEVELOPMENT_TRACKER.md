## Appendix: Git Commands Reference

### Check Status
```bash
git status                     # See modified/untracked files
git log --oneline -10          # See recent commits
git diff                       # See local changes
git diff origin/main           # Compare with remote
```

### Sync Operations
```bash
git pull origin main           # Pull remote changes
git push origin main           # Push local changes
git fetch origin               # Fetch without merging
git merge origin/main          # Merge remote into local
```

### Conflict Resolution
```bash
git status                     # See conflict files
git diff                       # See conflict markers
git checkout --theirs <file>   # Accept remote version
git checkout --ours <file>     # Accept local version
git add <file>                 # Mark conflict as resolved
git commit                     # Complete merge
```

### Stash Operations
```bash
git stash                      # Stash local changes
git stash list                 # List stashed changes
git stash apply                # Apply latest stash
git stash pop                  # Apply and remove latest stash
git stash drop                 # Remove latest stash
```

### Branch Operations
```bash
git branch                     # List local branches
git branch -a                  # List all branches
git checkout -b <branch>       # Create new branch
git checkout <branch>          # Switch to branch
git merge <branch>             # Merge branch into current
```
