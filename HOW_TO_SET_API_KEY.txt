METHOD 1: QUICK TEST (Temporary - 2 minutes)
═══════════════════════════════════════════════════════════════

Step 1: Get Your API Key
──────────────────────────
   1. Go to: https://platform.openai.com/api-keys
   2. Log in or create account
   3. Click "Create new secret key"
   4. Give it a name: "VerifiMind"
   5. Copy the key (starts with sk-...)

   Example key: sk-proj-abc123def456...


Step 2: Open Command Prompt
──────────────────────────
   1. Press Windows + R
   2. Type: cmd
   3. Press Enter

   [You'll see black window]


Step 3: Set the Key
──────────────────────────
   Type this (paste your real key):

   set OPENAI_API_KEY=sk-proj-your-actual-key-here

   Press Enter

   (Nothing will appear - that's normal!)


Step 4: Go to VerifiMind Folder
──────────────────────────
   Type this:

   cd "C:\Users\weibi\OneDrive\Desktop\VerifiMind Project 2025"

   Press Enter


Step 5: Launch VerifiMind
──────────────────────────
   Type this:

   python launch.py

   Press Enter

   [VerifiMind menu will appear]


Step 6: Verify It's Working
──────────────────────────
   In VerifiMind menu:

   Press [4] - Check System Status

   Should show:
   ✓ OpenAI: Configured


Step 7: Test with Real AI!
──────────────────────────
   Press [2] - Test Enhanced Agents

   Watch it use REAL GPT-4!


⚠️  IMPORTANT: This key only works in THIS Command Prompt!
    If you close it, you must set it again.



METHOD 2: PERMANENT (One-Time Setup - 5 minutes)
═══════════════════════════════════════════════════════════════

Step 1: Open System Properties
──────────────────────────────────
   1. Press Windows + R
   2. Type: sysdm.cpl
   3. Press Enter

   [System Properties window opens]


Step 2: Open Environment Variables
──────────────────────────────────
   1. Click "Advanced" tab (at top)
   2. Click "Environment Variables" button (at bottom)

   [New window opens]


Step 3: Create New Variable
──────────────────────────────────
   In "User variables" section (top half):

   1. Click "New" button
   2. Variable name: OPENAI_API_KEY
   3. Variable value: sk-proj-your-actual-key-here
      (Paste your real key!)
   4. Click OK


Step 4: Save Everything
──────────────────────────────────
   1. Click OK (Environment Variables window)
   2. Click OK (System Properties window)

   Done!


Step 5: Restart
──────────────────────────────────
   IMPORTANT: Close ALL windows and programs, then:

   Option A: Restart computer (safest)
   Option B: Just close all Command Prompts


Step 6: Verify It's Set
──────────────────────────────────
   1. Open NEW Command Prompt
   2. Type: echo %OPENAI_API_KEY%
   3. Should show your key!

   If it shows: %OPENAI_API_KEY%
   → Restart computer and try again


Step 7: Use VerifiMind Normally
──────────────────────────────────
   Now you can just double-click:

   VerifiMind.bat

   Your API key is always available!



TESTING YOUR API KEY
═══════════════════════════════════════════════════════════════

Test #1: Quick Check
──────────────────────────────────
   1. Open Command Prompt
   2. Type: echo %OPENAI_API_KEY%
   3. Should show your key (sk-proj-...)

   If it shows %OPENAI_API_KEY% → Key not set!


Test #2: VerifiMind System Check
──────────────────────────────────
   1. Double-click VerifiMind.bat
   2. Press [4] - Check System Status
   3. Look for:
      OpenAI: ✓ Configured

   If shows: ✗ Not set → Key not configured


Test #3: API Connection Test
──────────────────────────────────
   In VerifiMind folder, run:

   python test_api.py

   Should say:
   ✅ OpenAI API is working correctly!



TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Problem: Key not showing when I type echo %OPENAI_API_KEY%
──────────────────────────────────────────────────────────
   Solution:
   → Close ALL Command Prompts
   → Open NEW one
   → Try again
   → Still not working? Restart computer


Problem: VerifiMind still says "Not set"
──────────────────────────────────────────────────────────
   Solution:
   1. Verify key is set: echo %OPENAI_API_KEY%
   2. Make sure you closed and reopened Command Prompt
   3. Try Method 1 (temporary) first to test
   4. Then do Method 2 (permanent)


Problem: test_api.py says "Invalid API key"
──────────────────────────────────────────────────────────
   Solution:
   → Double-check you copied the FULL key
   → Key should start with sk-proj- or sk-
   → No spaces at beginning or end
   → Create new key on OpenAI if still fails


Problem: "Insufficient credits"
──────────────────────────────────────────────────────────
   Solution:
   → Go to: https://platform.openai.com/account/billing
   → Add payment method
   → Add $10+ credits
   → Wait 2 minutes for activation


Problem: Want to use Anthropic Claude instead?
──────────────────────────────────────────────────────────
   Same steps but use:

   Variable name: ANTHROPIC_API_KEY
   Variable value: sk-ant-your-anthropic-key

   Get key from: https://console.anthropic.com/
